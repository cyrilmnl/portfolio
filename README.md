# Portfolio

This project is a portfolio website built with Svelte and TailwindCSS.

|        | Version |
| ------ | ------- |
| Svelte | 5.0     |

### Local dev

```bash
# Install
npm install

# Run and build
npm run dev
npm run build

# format and lint
npm run format
npm run lint
```
